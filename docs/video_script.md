## 录制建议

1. **预热数据**：提前用生成的 94 天伪数据，确保成就已解锁、统计有内容
2. **BGM 音量**：录制时 BGM 调至 30% 左右，后期配音时降低到 10-15% 作为背景
3. **鼠标操作**：操作流畅自然，不要犹豫。每个点击控制在 0.5 秒内完成
4. **抽卡环节重点拍**：丝带拖动是最出彩的视觉亮点，给足 7-10 秒
5. **画面切换**：用 OBS 录制，切页面时不要停顿，一气呵成
6. **分辨率**：建议录制 1920×1080，确保文字清晰
