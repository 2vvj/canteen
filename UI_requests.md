# UI

风格参考可以想象为带有手绘插画的现代艺术市集或独立咖啡馆的视觉系统。

## 核心视觉设计语言（基于全新参考图）

背景与配色： 拒绝死板的单调灰。界面背景应该使用有色彩倾向的、低饱和度的“彩色纸张感”色块。例如：淡米黄（#FDFBF7）、复古浅蓝（#D0E3EF）、芥末绿（#F2E9D4）或复古红褐（#C86A5A）。各区域采用大面积的纯色块撞色，但整体色调保持柔和、不刺眼。

线条质感： 边框、分割线必须是**不完美、粗细不均、带有毛刺或排线感（Sketchy & Scratchy）**的手绘墨水线条。线条颜色多为深石墨色（#2B2B2B）或深褐色，避免纯黑。

图形与卡片： 所有的矩形容器或按钮都不能有完美的几何直角或绝对正圆的圆角。它们应该像是在纸上随手剪下来的色块，带有轻微的歪斜、不对称或波浪状的随性边缘（Organic Shapes）。

## 组件交互设计（打破常规的艺术感按钮）

按钮 1（独立手绘风）： 按钮的边框由多条重叠、错位的细线条组成（模仿反复描摹的效果）。

按钮 2（阴影错位风）： 按钮由底层的深色纯色块（作为阴影）和上层的有色卡片错位叠加而成。悬停（Hover）或点击（Pressed）时，上层卡片产生向阴影方向下压的平移物理动画，像真正的纸质贴纸被按压一样。

## 技术实现路径（Qt & C++）

高级自定义绘制 (重写 paintEvent & 使用 QPainterPath)：

不要使用常规的 painter.drawRect。请通过 QPainterPath 结合贝塞尔曲线（quadTo / cubicTo）或者在各顶点加入随机微小偏移（像素级抖动），来绘制“不完美”的矩形或圆形容器。

利用 QPen 的样式（如 Qt::DashLine 的变体，或通过多层微弱位移的线条叠加）来模拟墨水在纸张上的晕染与毛刺感。

动态重绘反馈： 当鼠标 hover 或 click 控件时，触发 update()，让随机线条的振幅产生微妙的变化，赋予界面生命力。

解耦设计： 提供干净的 C++ 类结构。
